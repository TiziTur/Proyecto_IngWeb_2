Acerca de la Materia

¿Qué aprenderemos?

Base de Datos II es una materia avanzada que profundiza en los conceptos introducidos en Base de Datos I. En este curso, nos enfocaremos en técnicas de optimización, administración de sistemas de bases de datos, seguridad de datos y diseño de bases de datos a nivel empresarial.

Desarrollarás habilidades prácticas para trabajar con bases de datos complejas, aprenderás sobre indexación, replicación, backup, recuperación y administración de servidores de bases de datos.

Requisitos Previos

  • ✓ Base de Datos I completado
  • ✓ Conocimiento de SQL básico
  • ✓ Conceptos de entidades y relaciones
  • ✓ Experiencia con al menos un SGBD

Temas Principales

📊 Diseño Avanzado

  • Normalización (hasta 5NF)
  • Desnormalización
  • Modelado dimensional
  • Data warehouse

⚡ Optimización

  • Índices y estrategias
  • Análisis de planes de ejecución
  • Queries optimization

🔐 Seguridad

  • Encriptación de datos

  • Cumplimiento normativo

💾 Administración

  • Backups y recuperación
  • Replicación de datos
  • Clustering

📈 Escalabilidad

  • Sharding
  • Caching distribuido
  • Load balancing

🌐 NoSQL y Modernos

  • Bases de datos NoSQL
  • MongoDB y similares
  • Time-series databases
  • Graph databases
